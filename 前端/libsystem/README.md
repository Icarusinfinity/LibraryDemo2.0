# 图书馆管理系统前端

基于Web的图书馆管理系统前端，实现图书管理、用户管理、借阅管理和统计报表等功能。

## 功能特性

- **图书管理**：图书入库、图书查询、图书编辑、图书删除
- **用户管理**：用户注册、用户登录、用户信息管理
- **借阅管理**：借书、还书、续期、借阅记录查询
- **统计报表**：借阅排行榜、用户借阅统计、图书分类统计
- **响应式设计**：适配不同屏幕尺寸
- **图表展示**：使用Chart.js展示统计数据

## 运行方法

### 方法一：使用Python HTTP服务器

```bash
cd libsystem
python -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`

### 方法二：使用Node.js live-server

```bash
npm install -g live-server
cd libsystem
live-server
```

然后在浏览器中访问显示的URL

## 主要功能模块

### 1. 控制台
- 图书总数、用户总数、借阅总数、逾期图书统计卡片
- 借阅趋势图表
- 热门图书统计图表

### 2. 图书管理
- 图书入库（添加新图书）
- 图书查询（按书名、作者、ISBN搜索）
- 图书分类筛选
- 图书编辑和删除

### 3. 用户管理
- 用户注册
- 用户登录
- 用户信息管理
- 用户类型管理（读者/管理员）

### 4. 借阅管理
- 借书功能
- 还书功能
- 借阅记录查询
- 续期功能

### 5. 统计报表
- 借阅排行榜（饼图）
- 用户借阅统计（柱状图）
- 图书分类统计（饼图）

## 响应式设计

系统支持桌面端和移动端访问，适配不同屏幕尺寸。

## 浏览器兼容性

支持现代浏览器，包括Chrome、Firefox、Safari、Edge等。
